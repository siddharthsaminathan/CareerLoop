There are ONLY 5 patterns.

# PATTERN 1 — PUBLIC API

BEST.

Example:
Naukri XHR.

Flow:

```text id="x7m2vr"
browser
→ intercept API
→ replay JSON
```

---

# PATTERN 2 — SEARCH INDEX SCRAPING

Example:
Glassdoor.

Flow:

```text id="m4q9tz"

→ job URL extraction
→ open page
→ extract JD
```

---

# PATTERN 3 — RENDERED DOM

Example:
LinkedIn.

Flow:

```text id="f8n1wc"

→ render
→ extract cards
```

Example:
Google Jobs.

Flow:

```text id="h2r7pv"

```

VERY powerful.
